# scanPaper
Esta app escanea documentos y lo procesa obteniendo lo contornos del documento, brindando una salida mas limpia de este.

## Pre-Requisitos
para poder compilar esta app, instale las siguiente librerias

```
python3

numpy

opencv

matplotlib

scipy

```

## Compilacion
```
python main.py

```
